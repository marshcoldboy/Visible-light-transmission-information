# 编码

### 介绍

编码的步骤基本如下：

1. 利用Buffer类来读取二进制
2. 利用QRCode模块来进行生成二维码
3. 利用Video模块来生成视频
4. 利用Encode模块来组织上述三个模块，实现编码

### 说明

1. 因为采用32位寻址，最大传输文件为4GB。
2. 传入输出文件参数的时候，无需加格式名，将会自动以avi格式输出
